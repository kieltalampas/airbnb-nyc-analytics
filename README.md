# NYC Airbnb Analytics Dashboard

## Project Overview
Interactive analytics dashboard analyzing 41,697 NYC Airbnb listings using Snowflake and Power BI.

## Data Source
- Dataset: Airbnb NYC 2019 (Kaggle)
- Records: 41,697 listings
- Key Metrics: Price, Reviews, Room Type, Neighborhood

## Tech Stack
- **Data Warehouse:** Snowflake
- **BI Tool:** Power BI
- **Data Source:** CSV (Kaggle)
- **Data Processing:** Power Query, SQL

## Dashboard Features

### Page 1: Executive Summary
- Total Listings (41,697)
- Average Price per Night ($143.74)
- Average Reviews per Listing (23)
- Price Distribution by Category (Pie Chart)

### Page 2: Neighborhood Analysis
- Total Listings by Neighborhood (Bar Chart)
- Average Nightly Price by Neighborhood (Bar Chart)
- Interactive Neighborhood Filter

### Page 3: Room Type & Price Analysis
- Average Reviews per Room Type (Bar Chart)
- Total Listings by Room Type (Pie Chart)
- Price vs. Number of Reviews (Scatter Plot)
- Interactive Room Type & Price Category Filters
